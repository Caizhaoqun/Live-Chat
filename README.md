Live-Chat
==============

### 运行系统
- Linux(推荐)
- Windows7/Windows8

### 运行环境  
- node: `v >= 0.10.32`
	- [安装](http://nodejs.org/)
	- 测试是否安装成功: `node -v`
- MongoDB: `v >= 2.4.9`
	- [安装](http://www.mongodb.org/downloads)
	- 测试是否安装成功： `mongo`(Linux)

### 运行命令
- `cd Live-Chat`
- `npm install`
- `grunt`
